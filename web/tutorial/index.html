Tutorial Demo

I developed a component that combines functionality of a slideshow gallery, a modal box, and a register form. The JS function uses a parameters object with the following fields:

  • id: the id of the div that the function inserts the component into
  • list: the list that the component uses to display images and captions

The JS function creates a component where the user can see a large picture with its corresponding caption. Below, there are smaller, less opaque images that are populated from the list parameter. When a user clicks on a smaller picture, that image and caption are displayed where the large image is. The user can also navigate to the next or previous images in the list with corresponding arrows. When a user clicked on the larger image, a modal will pop up with a form to purchase a ticket to that resort. For example, if the larger picture is a picture of the ski resort, Wisp, a modal will pop up prompting the user to purchase a ticket to Wisp.
